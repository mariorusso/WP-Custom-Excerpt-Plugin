Custom Excerpt allows you to costumize your excerpt length, read more text and add a link to it.

== Description ==

Allows admin user to specify length of excerpt in either words or characters, don't break words in the middle. 
Allows admin user to set the read more text for what they want.
Allows admin user to chose if want a link to the post or not in the read more text. 

== Installation ==

Upload the Custom Excerpt plugin to your worpress plugins folder.
Activate the plugin in the admin pannel. 
Use the link on the admin menu and set your options.

You are done. Easy!!!
